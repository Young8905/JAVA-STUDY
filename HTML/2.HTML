02-1 프로그램에서 입력하고 출력하는 방법

alert(내용)

  • 소괄호 안에 메세지를 입력하거나 변수를 사용하여 알림창에 표시
  • 내용표시 : 큰따옴표나 작은따옴표와 함께 내용을 작성 ex. alert("안녕하세요?")
  • 변수표시 : 변수이름을 지정하여 변수의 값을 표시 ex. a =10, alert(a)

    • confirm(내용)

  • <확인>, <취소> 버튼이 있어 사용자가 어떤 버튼을 클릭했는가에 따라 다르게 동작
  • ex. confirm("프로그램을 종료하시겠습니까?") : 취소를 누를경우 false가 동작됨

    • prompt(내용)

  • 사용자가 간단한 값을 입력할 수 있는 창
  • 사용자가 입력한 값을 가져와 프로그램에서 처리
  • ex. prompt("이름을 입력하세요.")
  • 기본값지정시 : prompt(내용,기본값), ex. prompt("참석여부(예/아니오)","예") 를 할 경우 기본적으로 예가 작성 되있음
  • 사용자가 취소버튼 클릭 시 null이 반환됨

    • console.log(내용)

  • 소괄호 안의 내용을 콘솔창에 표시할 때 사용
  • 자바스크립트 소스를 작성하며 중간에 프로그램이 제대로 동작하는지 확인하는 용도로 자주 사용
  • console.log("내용"), console.log(변수)로 입력
  • 변수와 내용을 같이 작성도 가능 console.log(변수 + "내용")

    • document.write(내용)

  • 소괄호 안의 내용을 웹 브라우저 창에 표시할 때 사용

    변수 이름 지정하기

  • 변수를 서로 구별할 수 있도록 변수마다 다른 이름을 붙여 주어야 함
  • 1. 변수 이름은 숫자로 시작할 수 없고 이름 안에 공백이 포함되어 있으면 안됨
  • 2. 영문자의 대소문자를 구별하여 사용 ex. current, Current, CURRENT는 모두 다른 이름
  • 3. 한 단어로 이루어진 변수를 사용할 때는 영문자의 소문자를 사용
  • 4. 두 단어 이상으로 이루어진 변소를 사용할때는 언더바로 연결하거나 중간에 영문자의 대문자를 섞어 사용 ex. total_current or totalCurrent(카멜표기법)
  • 5. let 등 자바스크립트에서 미리 정해 놓은 예약어는 변수 이름으로 사용 불가능

    • 변수 선언하기

  • 자바스크립테에는 값을 변경할 수 있는 "변수" 와 값이 바뀌지 않는 "상수" 가 있음
  • 변수라면 예약어 let 변수명 사용
  • 상수라면 예약어 const 변수명 사용

    • 변수에 값 할당하기

  • 사용할 변수를 선언하고 나면 = 기호를 사용해 변수에 값을 지정할 수 있음
  • 변수를 선언한 후 값을 따로 선언할 수도 있고, 변수를 선언하면서 값을 동시에 할당 할 수도 있음 ex. let sum; sum = 10; or let sum = 10;
  • let을 사용하여 변수를 선언한 후 그 변수명에 다시 값을 재할당하면 새로운 값이 저장이 됨
  • const를 사용하여 변수를 선언한 후 그 변수명에 다시 값을 재할당하면 오류가 발생

    • 변수를 선언할 상수가 필요한 이유

  • 프로그램에서 같은 값을 가져와 여러번 사용할 경우 아예 상수에다가 값을 할당해 놓고 가져와서 사용

    • var와 변수호이스팅

  • 호이스팅이란 자바스크립트해석기가 함수소르를 훑어보면서 변수는 따로 기억해 두는것
  • var 를 사용할 경우 호이스팅되어 오류가 나도 찾기가 어려워져 사용하지 않는 게 좋음
  • let 이나 const 를 사용할 경우에는 변수를 선언하기 전에 사용하면 오류가 발생하여 오류를 찾기 쉬움

    • 변수 재선언하고 재할당하기

  • var : 선언하지 않고 사용할경우 : 오류없음(호이스팅발생), 재선언과 재할당이 가능
  • let : 선언하지 않고 사용할경우 : 오류 발생, 재할당만 가능
  • const : 선언하지 않고 사용할경우 : 오류발생, 재선언, 재할당 둘다 불가능

    • 02-3 컴퓨터가 세상을 보는 방법, 자료형

    typeof(내용)

  • 소괄호 안에 값이나 변수를 넣으면 어떤 자료형인지 알려줌

    • 숫자형(number)

  • 모든 프로그램에서 가장 기본이 되는 자료형
  • 큰 따옴표 없이 입력하면 숫자로 인식

    • 문자형(string)

  • 작은따옴표'' 큰따옴표""로 묶은 데이터
  • 숫자도 따옴표를 사용하여 입력할 경우 문자형으로 인식

  • 특수기호 표시하기 (백슬래시(|)이용)
  • ex. console.log('I|'m studying now.') - I다음 작은따옴표를 표시하기 위해 백슬래시를 이용하여 작성

  • 템플릿 리터럴 : 문자열과 변수 식을 썩어서 하나의 문자열을 만드는 표현 형식
  • 백팃(``)을 사용하여 문자열을 만드는 표현 형식
  • 템플릿 리터럴 안에 변수나 식이 들어간다면 ${}로 묶어서 사용
  • ex. console.log(`${변수명}, "내용"`) : 백킷 기호와 내용및 변수를 사용하여 작성

    • 논리형(boolean)

  • 참(true)이나 거짓(false)의 값을 표현하기 위한 데이터 유형, 조건식에서 많이 사용
  • false로 인정되는 값 : 0, ""(빈문자열), NaN(숫자가아님), undefined, null0

    • undefined, null

  • undefined : 값이 할당되지 않았을때 변수의 초깃값, 빈 값인지 유효한 값인지 정할 수 없는 상태 ex. 변수를 선언만 하고 값을 할당하지 않은 상태에서 출력시 반환
  • null : 값이 없거나 유효하지 않은 값,

    • 객체(object)

  • 하나의 변수에 다양한 정보가 포함되어 있는 자료형
  • 중괄호({})안에 키:값을 이용하여 정보를 담음
  • 작성방법 : 객체명 = { 키:값, 키:값, ...}
  • 키는 문자열이어도 큰따옴표를 사용하지 않음
  • 객체의 값에 접근하기 : 객체명.키 이름 or 객체명["키 이름"] 입력

    • 배열(array)

  • 하나의 변수에 여러 개의 값을 저장할수 있는 유형
  • 대괄호([])안에 값을 나열하여 입력
  • 작성방법 : 배열명 = [값, 값, 값,...]
  • 배열값의 순서를 인덱스(index)라고함, 0부터 시작
  • 배열에 모두 몇개의 값이 있는지 알고싶으면 length 사용, ex. 배열명.length

    • 심벌(symbol)

  • 유일성을 보장, 객체 프로퍼티의 키로 사용할수 있음
  • 다른 사람과 만든 객체를 함께 사용하여 키이름이 중복될 경우 심벌로 지정하면 충돌이 생기지 않음
  • Symbol()함수를 사용하여 만듬, 한번 만들면 변경할 수 없고 같은 값을 가진 심벌을 만들수도 없음
  • 심벌을 키로 사용할 경우 대괄호([키])로 묶어서 표현
  • 심벌로 지정한 키에 다른 값을 재지정 할경우 심벌키는 그대로 유지되고 다른 값을 가진 키가 추가되어 키의 밧이 변경되는 것을 방지 할 수 있음
  • 변수명 = Symbol()로 작성 하나 괄호안에 설명을 넣어서 알아보기 쉽게 만들어 줄 수 있음

    • 02-4 자바스크립트의 장점이자 단점, 자료형 변환

    자바스크립트 자료형의 특징

  • 미리 변수의 자료형을 지정하지 않음
  • 변수에 값을 할당하면 그 값에 따라 자료형이 결정
  • 같은 변수에 다른 유형의 값을 잘못 입력해도 체크할 수 없고 여러사람이 프로젝트를 진행할 경우 변수를 일관성 있게 유지하기가 어려움
  • 문자열을 사칙연산에 사용할 때는 자동으로 숫자형으로 변환
  • 숫자와 문자열을 연결하면 숫자가 자동으로 문자형으로 변환

    • 직접 숫자형으로 변환하기

  • 숫자가 아닌 값을 숫자로 변환할 때 사용
  • ex. 프롬프트창에서 입력받은 숫자는 문자열로 취급 이때 숫자형으로 바꾸기 위해 사용
  • Number() : 소괄호 안의 값을 숫자로 변환
  • parseInt() : 소괄호 안의 값을 정수로 변환
  • parseFloat() : 소괄호 안의 값을 실수로 변환

    • 문자열로 변환히기

  • 문자열이 아닌 값을 문자열로 변환할 때 사용
  • toString() : null 과 undefined를 제외한 나머지 자료형을 문자열 데이터로 변환, 작성법 : 값.toString()
  • 숫자를 문자열로 변환할 경우에는 기본값을 사용해 10진수인지 2진수 인지 지정할 수 있음, 작성법 : 값.toString(기본값 또는 2), 기본값을 지정하지 않으면 10진수로 표현
  • String() : null 과 undefined까지 모두 포함해서 문자열 데이터로 변환 작성법 : String(값)
  • * 값의 앞에 + 기호를 붙이면 자동으로 문자열이 숫자로, 숫자에 빈 문자열을 연결하면 숫자가 문자열로 변환 *

    • 논리형으로 변환하기(Boolean)

  • 작성법 : Boolean(값)
  • 값이 0이거나 비어있으면 false 로 그 외의 값은 true 로 변환됨
